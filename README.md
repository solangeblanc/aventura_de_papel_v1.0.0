# 🎮 Aventura de Papel - Edición Original (v1.0.0)

Esta es la primera versión estable de **Aventura de Papel**, un proyecto que explora la narrativa visual a través de un estilo artístico artesanal y mecánicas de plataformas clásicas.

## ✒️ Concepto de Diseño
El objetivo principal de esta versión fue lograr una estética de "cuaderno de bocetos" (sketchbook) cobrando vida:
* **Visuales:** Arte dibujado a mano alzada.
* **Tipografía:** [Pixellari](https://www.dafont.com/pixellari.font) para un contraste digital/analógico.
* **Interfaz (UI):** Menús minimalistas y directos para priorizar la jugabilidad.

## 🚀 Estado del Proyecto
* **Motor de juego:** Construct 2.
* **Despliegue:** Hosting gratuito vía GitHub Pages.
* **Compatibilidad:** Optimizado para navegadores de escritorio.
